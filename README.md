# 🚺 AI Safety Companion Web App

An **AI-powered Women Safety Web Application** designed to assist users in unsafe situations by providing real-time guidance, emotional reassurance, and emergency support.

The system simulates a **phone call safety assistant**, guides users to nearby safe places, and allows quick emergency alerts.

---

# 💡 Problem Statement

Women often face unsafe or uncomfortable situations in public spaces such as isolated streets, late-night travel, or unfamiliar areas.

In such situations:
- Panic makes it difficult to think clearly
- Calling someone may not always be possible
- Finding nearby safe places quickly is difficult
- Most safety apps only provide basic SOS alerts

There is a need for a **smart safety assistant** that can guide and support users in real time.

---
# 🚀 Solution

Our project introduces an **AI Safety Companion** that acts like a trusted person on a phone call during emergencies.

When a user feels unsafe, they can activate the **Safety Call Assistant**, which:

- Starts a simulated phone call
- Talks to the user using AI voice
- Understands the user's situation
- Suggests nearby safe or crowded places
- Guides the user step-by-step
- Sends alerts to emergency contacts

---

# ⭐ Key Features

## 🚨 SOS Emergency Alert
- Quickly send alerts to emergency contacts
- Share current location
- Notify trusted contacts during danger

---

## 📞 AI Fake Call Simulation
Simulates a real phone call to help the user appear busy.

Features:
- Incoming call screen
- Ringtone simulation
- AI voice conversation

Works **even without internet connection**.

---

## 🧠 AI Safety Assistant
An AI assistant that communicates with the user during stressful situations.

Example:

User:
> Someone is following me.

AI:
> Stay calm. Try moving toward a crowded place like a café or store.

---

## 📍 Smart Safe Place Guidance
The system identifies nearby safe places such as:

- Cafés
- Restaurants
- Hospitals
- Police Stations
- Shopping malls
- Bus stops

The assistant then guides the user toward safer areas.

---

## 🗺️ Real-Time Location & Map
- Detects user location using GPS
- Displays location on map
- Helps identify nearby safe locations

---

## 🔊 Voice Interaction
Supports hands-free interaction:

- Voice input from the user
- AI voice responses
- Natural conversational experience
